# Disease Prediction Model
This project predicts the possibility of a disease using patient medical data.

## Files
- disease_prediction_model.py : main script
- disease_data.csv : sample dataset
- requirements.txt : dependencies

## Run
1. python -m venv venv
2. source venv/bin/activate   # or venv\Scripts\activate on Windows
3. pip install -r requirements.txt
4. python disease_prediction_model.py
